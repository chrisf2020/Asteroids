## Asteroids Game
A classic arcade style space shooter built with Python. Navigate your ship through a dangerous asteroid field, blast obstacles, and survive as long as you can!

# Quick Start
This project uses uv for management. To run the game use:

"uv run main.py"

# How to Play
Controls
Rotate: A & D Keys
Forward: W Key
Backward: S Key
Shoot: Spacebar
Quit: ESCAPE

# Tech Stack
Language: Python
Dependency Manager: uv
Library: Pygame
